# cheeckout 结算页面

* 实现多类型，多家公司结算
* 多组件
* ui ant


### 使用的主要技术

* vue-router
* vuex
* axios
* vant
* base64加密


## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run all tests
npm test
```

> ps: 项目不可运行,部分代码已删除，内部项目
